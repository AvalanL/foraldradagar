Integritetspolicy

Senast uppdaterad: 25 februari 2026

Sammanfattning: Föräldradagar lagrar all din data lokalt på din enhet. Vi samlar inte in, säljer eller delar dina personuppgifter. Den enda datan som lämnar din enhet är frågor du ställer till AI-rådgivaren.

1. Vilka vi är

Föräldradagar är en iOS-app som hjälper svenska föräldrar att planera sin föräldraledighet. Appen utvecklas och drivs av Föräldradagar ("vi", "oss").

2. Data vi INTE samlar in

Vi samlar inte in följande data:

  • Namn, e-postadress eller kontaktuppgifter
  • Inkomstuppgifter eller ekonomisk information
  • Barnets födelsedatum eller familjeuppgifter
  • Platsdata eller IP-adress
  • Användningsstatistik eller analyser
  • Enhetsidentifierare eller spårningsdata

3. Lokal datalagring

All information du anger i appen — namn, inkomster, barnets födelsedatum, planerade ledighetsdagar — lagras enbart lokalt på din enhet med Apples SwiftData-ramverk. Datan synkas inte till någon server och lämnar aldrig din enhet (förutom vid AI-frågor, se nedan).

4. AI-rådgivaren

När du ställer en fråga till AI-rådgivaren skickas följande data till Anthropics API (Claude):

  • Din fråga (det du skriver)
  • Anonymiserad familjekontext (antal dagar kvar, inkomstnivå, barnets ålder) — inga namn skickas
  • Konversationshistorik för den aktuella sessionen

Anthropic behandlar denna data enligt sin integritetspolicy. Anthropic använder inte API-data för att träna sina modeller.

5. Köp och prenumerationer

Köp hanteras helt av Apple genom App Store. Vi har inte tillgång till dina betalningsuppgifter. Apple hanterar all betalningsinformation enligt Apples integritetspolicy.

6. Tredjepartstjänster

Appen använder följande tredjepartstjänster:

  • Anthropic Claude API — för AI-rådgivaren (se punkt 4)
  • Apple StoreKit — för köp och prenumerationer

Vi använder inga analys-, reklam- eller spårningsverktyg.

7. Barns integritet

Appen samlar inte medvetet in personuppgifter från barn. Appen är avsedd för föräldrar och vårdnadshavare. Barnets födelsedatum som anges lagras enbart lokalt för att beräkna ledighetsdagar och deadlines.

8. Dina rättigheter (GDPR)

Eftersom all data lagras lokalt på din enhet har du full kontroll:

  • Radera data: Avinstallera appen för att radera all data, eller rensa i appens inställningar
  • Exportera data: Appen lagrar data lokalt — du äger den helt
  • Rätt till information: Denna policy beskriver all databehandling

9. Ändringar i policyn

Vi kan uppdatera denna integritetspolicy vid behov. Ändringar publiceras på denna sida med nytt datum. Väsentliga ändringar meddelas i appen.
